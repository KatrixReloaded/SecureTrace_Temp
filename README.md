# SecureTrace  
### A Blockchain Forensics Tool to track crypto assets and transactions  
  
**Set up two variables in .env in the root directory**  
- ETHERSCAN_APIKEY  
- ALCHEMY_APIKEY  
- DB_HOST  
- DB_USER  
- DB_PASS  
- DB_NAME  
  
**Then run**  
```bash
npm i
source .env
node src/back-end/SecureTrace.js
npm start
```
