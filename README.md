# SecureTrace  
### A Blockchain Forensics Tool to track crypto assets and transactions  
  
**Set up two variables in .env in the root directory**  
- ETHERSCAN_APIKEY  
- ALCHEMY_APIKEY  
  
**Then run**  
```bash
npm i
source .env
node src/txTracingVisualizer/networks/SecureTrace.js
npm start
```
